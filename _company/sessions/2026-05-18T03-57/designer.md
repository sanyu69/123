# 🖼️ MVP 아트 에셋 최종 디자인 명세 (Basic / Pro 티어)

본 명세는 **'Neon Vibe'** 미학을 기반으로 하며, IAP 전환 흐름 최적화 및 수익 극대화를 위한 시각적 차별화를 목표로 합니다. 모든 색상 코드와 레이아웃 좌표는 개발자가 즉시 적용할 수 있도록 구체적으로 정의되었습니다.

## 1. 디자인 시스템 요약 (Neon Vibe)

*   **핵심 컬러 팔레트 (Neon\_Vibe\_Palette):**
    *   **Primary Accent (Pro/VIP 강조):** `#00FFFF` (Cyan/Electric Blue) - 최고 등급 시각적 강조에 사용.
    *   **Secondary Accent (Basic):** `#39FF14` (Neon Green) - 기본 기능 및 단계 표시용으로 사용.
    *   **Background/Base:** `#1A1A2E` (Deep Dark Blue/Violet) - 전체적인 네온 분위기를 극대화하는 배경색.
    *   **Text/Interface:** `#FFFFFF` (Pure White) 또는 `#EEEEEE` (Light Gray)

*   **타이포그래피:** 고정된 'Neon' 스타일의 산세리프 폰트 사용 권장 (예: Orbitron, Neon, 또는 유사한 네온 느낌의 폰트).

## 2. IAP 아트 에셋 상세 명세

### A. Basic 티어 아트 에셋 (기본 기능)

**목표:** 최소한의 시각적 차별화 및 기본 경험 제공.

| 요소 | 디자인 요구사항 | 색상 코드 | 레이아웃 좌표/규칙 |
| :--- | :--- | :--- | :--- |
| **전환 버튼 배경** | 부드러운 네온 그라데이션 효과 (Base: `#1A1A2E` -> Accent: `#39FF14`) | Base: `#1A1A2E`, Accent: `#39FF14` | 전체 화면 중앙 배치, 패딩 80px. |
| **프리미엄 배지** | 단순한 테두리 강조 (Outline) | 테두리: `#FFFFFF`, 내부: `#1A1A2E` | 아이콘 주변에 적용. |
| **레이아웃 흐름** | 선형적이고 명확한 진행 표시. | - | 1단계(Basic) 시각화 완료 후 다음 단계로 이동. |

### B. Pro 티어 아트 에셋 (핵심 차별화)

**목표:** 기능적 우위 및 'Neon Vibe'의 최대 강조.

| 요소 | 디자인 요구사항 | 색상 코드 | 레이아웃 좌표/규칙 |
| :--- | :--- | :--- | :--- |
| **전환 버튼 배경** | 강렬한 네온 그라데이션 (Base: `#1A1A2E` -> Accent: `#00FFFF`) | Base: `#1A1A2E`, Accent: `#00FFFF` | 전체 화면 중앙 배치, 패딩 80px. **(Primary Focus)** |
| **프리미엄 배지** | 네온 테두리와 내부 채우기 (Filled) | 테두리: `#00FFFF`, 내부: `#1A1A2E` | 아이콘 주변에 적용. |
| **레이아웃 흐름** | 복잡한 기능/고급 설정 접근 시, 더 밝은 명암 대비를 사용하여 '프리미엄' 느낌 강조. | - | 고급 설정 모달에서 사용. |

## 3. 수익화 흐름 최적화 (Monetization Flow) 아트 우선순위

수익 극대화를 위해 **Basic $\rightarrow$ Pro** 전환 흐름에 필요한 시각적 요소의 우선순위는 다음과 같습니다.

1.  **P1 (최우선):** Basic에서 Pro로 전환할 때 발생하는 **'가치 격차(Value Gap)'**를 명확히 보여주는 아트 에셋 (Pro 버튼 디자인 및 배지).
2.  **P2:** VIP 티어의 시각적 혜택을 강조하는 아트 에셋 (Exclusive Border, 특별 아이콘 등).
3.  **P3:** 일반적인 UI 요소 (메인 메뉴 배경, 텍스트 스타일).
