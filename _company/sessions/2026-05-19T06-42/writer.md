## ✨ 랜딩 페이지 카피 및 홍보 후크 아이디어 (3가지)

### 🚀 옵션 1: 시스템 안정성 강조형 (R_Stability 중심)

**🎯 타겟 심리:** 복잡한 게임 개발의 불안감을 해소하고, '안정적인 성장'이라는 확실한 결과물을 원하는 사용자.

| 구분 | 내용 |
| :--- | :--- |
| **🔥 후크 (Hook)** | **"개발 중 흔들리는 시스템? 24시간 안정된 성장의 비밀을 지금 경험하세요."** |
| **📝 본문 (Body)** | 수많은 기능과 복잡한 IAP 통합 과정에서 시스템 안정성($R_{Stability}$) 확보는 가장 큰 숙제입니다. 우리는 단순한 게임이 아닌, $R_{Stability}$와 $S_{Success}$를 수학적으로 설계하여 구현했습니다. 이 웹사이트는 개발 중에도 당신의 성장이 멈추지 않도록 보장하는 엔진입니다. |
| **📣 CTA (Call to Action)** | **"안정성 기반으로 시작하기: 지금 바로 게임 플레이 및 IAP 통합 경험하기"** |

---

### 🌌 옵션 2: 세계관 몰입 강조형 (Lore/Story 중심)

**🎯 타겟 심리:** 게임의 깊은 스토리와 분위기에 매료되어, 단순한 도구가 아닌 '경험'을 구매하고 싶은 사용자.

| 구분 | 내용 |
| :--- | :--- |
| **🔥 후크 (Hook)** | **"빛과 기억 속에서 깨어난 존재들. 당신의 선택이 세상을 바꿉니다."** |
| **📝 본문 (Body)** | 고대의 힘과 잊혀진 기억이 얽힌 세계, 네온 빛 아래 자동 액션으로 진화하세요. 단순한 클릭을 넘어, 당신의 성장이 곧 이 세계의 운명입니다. 깊은 몰입감과 아름다운 시각적 경험 속에서 새로운 차원의 존재로 거듭나십시오. |
| **📣 CTA (Call to Action)** | **"기억을 깨우고 힘을 얻으세요: 지금 바로 네온 서바이버 모험 시작!"** |

---

### 💡 옵션 3: 자동 성장 및 효율 강조형 (Automation/Efficiency 중심)

**🎯 타겟 심리:** 최소한의 노력으로 최대의 성과를 내고 싶은, 자율 학습 지향적인 사용자.

| 구분 | 내용 |
| :--- | :--- |
| **🔥 후크 (Hook)** | **"자동 액션 + 성장 시스템: 당신이 생각할 틈 없이 레벨업을 경험하세요."** |
| **📝 본문 (Body)** | 수동 조작의 한계를 넘어선 자동화된 성장의 세계. 복잡한 컨트롤은 필요 없습니다. 시스템이 스스로 움직이며 당신의 잠재력을 극대화합니다. 가장 효율적인 방법으로, 몰입감 넘치는 게임 경험을 즉시 시작하세요. |
| **📣 CTA (Call to Action)** | **"자동 성장 엔진 활성화: 클릭 한 번으로 무한 성장의 길 열기"** |
