## 🎬 유튜브 영상 기획 (3가지 주제)

### 1. 주제: 압도적인 비주얼의 힘 (The Power of Visuals)
**🎯 목표:** 게임의 시각적 혁신(GLOW/Effects)이 단순한 미학을 넘어 실제 플레이에 어떤 영향을 미치는지 직관적으로 보여주어, '차원이 다른 경험'이라는 가치를 각인시킨다.
**🎥 영상 컨셉:** 짧고 강렬한 액션 몽타주와 극적인 비주얼 전환을 중심으로 구성. (Before/After 대비 강조)

| 구분 | 내용 |
| :--- | :--- |
| **핵심 메시지** | "당신의 실력이 시각적으로 증명된다. 단순한 게임이 아니다, 차원이 다른 경험이다." |
| **최적의 썸네일 카피 (Hook)** | **[🔥경고] 이 비주얼, 당신의 실력을 증명한다! (GLOW/Effect 체험)** |
| **보조 문구** | 등급 상승의 비밀? 시각 효과가 게임을 바꾼다. |

### 2. 주제: 명예와 성취의 과정 (The Path to Status)
**🎯 목표:** 플레이어가 성장하고 등급이 오르는 과정을 '명예 획득'이라는 심리적 보상과 연결하여 몰입감을 극대화한다. IAP 구매가 단순한 기능 해제가 아닌, **사회적 인정**을 얻는 행위임을 강조한다.
**🎥 영상 컨셉:** 캐릭터의 레벨업/등급 상승 순간에 시각 효과(GLOW)가 폭발하는 클로즈업 장면과 함께 성취감을 극대화하는 사운드 디자인 집중.

| 구분 | 내용 |
| :--- | :--- |
| **핵심 메시지** | "노력은 명예로, 당신의 실력은 빛난다. 등급 상승이 가져오는 압도적인 권한." |
| **최적의 썸네일 카피 (Hook)** | **[👑등급 상승] 이 순간, 당신은 왕이다! (VIP 비주얼 해금)** |
| **보조 문구** | 지위와 명예를 시각적으로 증명하는 시스템. |

### 3. 주제: 프리미엄 경험의 차이 (The Premium Experience)
**🎯 목표:** 무료 사용자 경험과 유료(프리미엄) 사용자 경험 사이의 극명한 차이를 대비시켜, 유료 전환의 필요성과 가치를 명확하게 제시한다.
**🎥 영상 컨셉:** 무료 버전과 프리미엄 버전을 화면 분할(Split Screen)하여 시각적 차이(GLOW/Effects의 강도 및 디테일)를 직접 비교하며 보여주는 방식.

| 구분 | 내용 |
| :--- | :--- |
| **핵심 메시지** | "무료는 시작일 뿐, 진정한 경험은 프리미엄에서 완성된다. 당신의 선택이 차이를 만든다." |
| **최적의 썸네일 카피 (Hook)** | **[💰차원이 다른 경험] 무료 vs VIP, 시각적 차이가 승패를 가른다** |
| **보조 문구** | 왜 프리미엄 유저들은 이 비주얼을 선택하는가? |
