## 💎 웹사이트 핵심 시각 디자인 가이드라인 (Final Visual Specification)

이 가이드라인은 모든 페이지에서 일관된 **Neon Vibe**와 **Dynamic System Proof** 미학을 구현하여 사용자 신뢰도를 극대화하고 IAP 전환을 유도하는 것을 목표로 합니다.

### 1. 브랜드 컬러 시스템 (Color System: Neon Vibe & Stability)

컬러 팔레트는 안정성($R_{Stability}$) 데이터에 따라 동적으로 변화하며, 기본 상태는 네온 테마를 유지합니다.

| 요소 | 역할 | 색상 코드 (HEX) | 설명 및 적용 규칙 |
| :--- | :--- | :--- | :--- |
| **Primary Neon** | 메인 강조색 (Dynamic Effects) | `#00FFFF` (Cyan) | 시스템이 '안정 상태'일 때의 기본 활성화 색상. 모든 동적 효과의 중심. |
| **Stability Green** | 안정성 지표 (Success/Stable) | `#00FF7F` (Spring Green) | $R_{Stability}$가 높을 때 표시되는 긍정적인 신호. |
| **Warning Yellow** | 경고/주의 (Dynamic Effect Trigger) | `#FFFF00` (Yellow) | 시스템이 '경고 상태'에 있거나, Dynamic Effects가 활성화될 때 사용. |
| **Critical Red** | 위험/불안정 (Failure/Risk) | `#FF4500` (OrangeRed) | $R_{Stability}$가 낮을 때 표시되는 즉각적인 주의 신호. |
| **Background Base** | 메인 배경 | `#1A1A2E` (Deep Navy/Dark Purple) | 네온 효과를 극대화하는 어둡고 깊은 배경. |
| **Text Primary** | 주요 텍스트 | `#FFFFFF` (White) | 가독성을 위해 밝은 색상 사용. |

### 2. 타이포그래피 시스템 (Typography System: Futuristic & Clean)

폰트는 기술적이고 미래 지향적인 느낌을 주며, 데이터의 명확성을 강조합니다.

*   **Font Family:** `Orbitron`, `Space Mono` 또는 `Bebas Neue` (선택적으로 사용)
    *   **Primary Header (H1, H2):** `Orbitron` (강렬하고 미래적인 느낌)
    *   **Body Text & Data Labels:** `Space Mono` (데이터의 정확성과 기술적 신뢰성 강조)
*   **Text Weight:** Bold, Light 사용을 명확히 구분하여 정보 계층 구조를 만듭니다.

### 3. 메인 페이지 레이아웃 및 CTA 디자인 (Homepage Layout & CTAs)

메인 페이지는 **'신뢰도 증명 > 가치 제시 > 행동 유도'**의 흐름을 따라 설계합니다.

#### A. 헤더 영역 (Header Area)
*   **디자인:** 최소화된 미니멀리즘. 배경은 `#1A1A2E`.
*   **핵심 요소:** 로고(123)와 간결한 내비게이션. 상단에 **$R_{Stability}$의 실시간 시각화 바 (Trust Bar)**를 고정 배치합니다.

#### B. 핵심 비주얼 섹션 (Hero Section: The Proof)
*   **목표:** 방문자가 스크롤하는 순간, 시스템의 신뢰성을 즉시 인지하게 합니다.
*   **레이아웃:** 화면 중앙에 **가장 크고 역동적인 Dynamic Effects**를 적용한 게임 플레이 스틸컷 또는 짧은 루프 애니메이션을 배치합니다.
*   **CTA 배치:** 시각적 증명 바로 아래에 IAP 전환 버튼을 배치합니다.

| 요소 | 디자인 세부 사항 | 목적 |
| :--- | :--- | :--- |
| **Trust Bar (상단 고정)** | 화면 상단에 Cyan/Yellow/Red 색상이 $R_{Stability}$ 수치에 따라 실시간으로 깜빡이며 움직입니다. | 시스템 안정성을 즉각적으로 시각화하여 신뢰 구축. |
| **Main CTA Button** | `Pro Upgrade Now` 또는 `Unlock Dynamic Proof` 버튼. 배경은 `#00FFFF`, 텍스트는 `#1A1A2E`. 마우스 오버 시, 버튼이 Cyan으로 빛나며 미세한 떨림(Glitch effect)을 추가합니다. | 행동 유도 및 '프리미엄' 경험 강조. |

#### C. 기능/가치 섹션 (Features Section: The Vibe)
*   **디자인:** 각 기능은 **Neon Vibe**의 구체적인 시각적 예시와 함께 제시되어야 합니다.
*   **레이아웃:** 3분할 또는 그리드 레이아웃을 사용합니다.
    1.  **Stability Visualizer:** (Trust Bar의 상세 설명) $R_{Stability}$ 데이터가 어떻게 안정성을 보여주는지 다이어그램으로 설명.
    2.  **Dynamic Effect Showcase:** 실제 게임 내에서 적용되는 네온 및 동적 효과를 고화질 캡처 이미지로 제시.
    3.  **System Proof Explanation:** 기술적 안정성이 사용자 경험에 미치는 영향을 짧은 문장으로 요약.

### 4. 최종 디자인 명세 (Summary for Developer)

개발팀에게 전달할 핵심 규칙입니다.

1.  **Dynamic System Rule:** 모든 움직임, 색상 변화, UI 상호작용은 **$R_{Stability}$ 수치**에 직접적으로 매핑되어야 합니다.
2.  **Color Mapping:** $R_{Stability}$ 값의 범위에 따라 Trust Bar의 색상이 다음 규칙을 따릅니다:
    *   $95\% \sim 100\%$: `Stability Green` (`#00FF7F`) - 안정
    *   $80\% \sim 95\%$: `Primary Neon` (`#00FFFF`) - 주의 (Dynamic Activation)
    *   $<80\%$: `Critical Red` (`#FF4500`) - 경고 (System Risk)
3.  **Aesthetic:** 배경은 어둡게 유지하고, 모든 주요 정보와 강조 요소는 네온 계열의 색상으로 **빛나도록(Glow)** 처리하여 'Neon Vibe'를 극대화합니다.
