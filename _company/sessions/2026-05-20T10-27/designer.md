## 📝 최종 시각 자료 디자인 명세서 (Visual Asset Specification)

본 명세서는 **'Neon_Vibe_Palette'** 및 **'Trust Bar 규칙'**을 모든 마케팅 콘텐츠에 적용하는 것을 최우선 목표로 합니다.

### 1. 브랜드 컬러 시스템 (The Neon Vibe Palette)

모든 시각 자료는 다음의 네온 기반 팔레트를 엄격하게 준수해야 합니다.

| 역할 | 색상 코드 (HEX) | 용도 | 적용 방식 |
| :--- | :--- | :--- | :--- |
| **Primary Glow (주요 빛)** | `#00FFFF` (Cyan/Neon Blue) | 핵심 강조, Dynamic Effects의 주 색상 | 배경 하이라이트, 주요 텍스트 테두리 |
| **Secondary Accent (보조 강조)** | `#FF00FF` (Magenta/Neon Pink) | 대비 효과, CTA 버튼, 위험 신호(안정성 경고 시) | 강조 요소, Dynamic Effects의 보조 색상 |
| **Background Base (배경 기반)** | `#0A0A1A` (Deep Navy/Black) | 전체 배경, 몰입감 극대화 | 기본 배경색 |
| **Stability Indicator (안정성 지표)** | `#39FF14` (Neon Green) | $R_{Stability}$ 데이터 시각화 | 안정성 수치 표시, 긍정적 피드백 |

### 2. Trust Bar 규칙 (신뢰도 증명 시스템)

모든 영상 썸네일과 피드 이미지에는 신뢰도를 즉각적으로 전달하는 'Trust Bar' 요소를 반드시 포함해야 합니다.

*   **위치:** 화면의 **좌측 하단 모서리**에 고정 배치합니다.
*   **디자인:** 배경은 어두운 네온 블루 계열(`#0A0A1A` 기반)로 처리하며, 텍스트는 `Primary Glow (#00FFFF)` 색상을 사용합니다.
*   **내용 구성 (3단계):**
    1.  **Step 1: 안정성 증명:** $R_{Stability}$ 수치 표시 (Neon Green `#39FF14`로 강조).
    2.  **Step 2: 핵심 가치:** 'System Stable' 또는 'Verified Flow' 등의 문구 삽입.
    3.  **Step 3: 행동 유도:** 작은 화살표 아이콘과 함께 "Proof Inside" 또는 "View Data"를 배치하여 클릭을 유도합니다.

### 3. 콘텐츠별 최종 비주얼 명세

#### A. YouTube 썸네일 (핵심 목표: 안정성 입증 - 영상 1)

| 요소 | 명세 | 적용 규칙 |
| :--- | :--- | :--- |
| **배경** | `#0A0A1A` (Deep Navy/Black) 기반, 미세한 네온 블루 그라데이션. | Background Base |
| **중앙 오브젝트** | 거대한 `$R_{Stability}$ 수치`를 중앙에 배치합니다. | `Primary Glow (#00FFFF)` 또는 `Stability Indicator (#39FF14)`로 처리하여 시각적 충격을 극대화합니다. |
| **Dynamic Effects** | $R_{Stability}$ 숫자를 중심으로 미세한 네온 효과(GLOW)를 적용하여, 수치가 살아 숨 쉬는 듯한 느낌을 연출합니다. | `Primary Glow` 및 `Secondary Accent`의 은은한 블렌딩 사용. |
| **Trust Bar** | 좌측 하단에 정의된 Trust Bar 규칙을 정확히 삽입합니다. | 신뢰도 확보 (필수) |
| **폰트 스타일** | 모던하고 미래지향적인 산세리프체 (예: Orbitron, Exo 2 등). 모든 텍스트는 네온 효과를 갖습니다. | Neon Vibe |

#### B. Instagram 피드 이미지 (핵심 목표: 시각적 차별화 - 영상 2)

| 요소 | 명세 | 적용 규칙 |
| :--- | :--- | :--- |
| **비주얼 컨셉** | GLOW 및 Dynamic Effects를 극대화한 게임 플레이 순간(슬로우 모션 또는 루프). 빛의 흐름과 움직임에 집중합니다. | 시각적 차별화 (GLOW/Dynamic Effects) |
| **컬러 톤** | `#0A0A1A` 배경 위에서, `Primary Glow (#00FFFF)`와 `Secondary Accent (#FF00FF)`를 대비시켜 역동성을 부여합니다. | Neon Vibe Palette |
| **텍스트 오버레이** | 핵심 카피(`GLOW & Dynamic Effects`)는 중앙에 배치하며, 폰트는 선명하고 간결해야 합니다. | 명확성 확보 |
| **Trust Bar** | Instagram 피드에서는 Trust Bar를 이미지의 **하단 1/5 지점**에 배치하여 콘텐츠 영역을 침해하지 않도록 조정합니다. | 신뢰도 유지 (필수) |
