## 🚀 게임 웹사이트 완성 로드맵 (Stability & UX First)

본 로드맵은 **기능 구현 → 안정성 검증 → 최적화 및 출시 준비**의 3단계 흐름으로 구성됩니다. 모든 마일스톤은 이전 단계의 기술적 결함을 다음 단계에서 재발하지 않도록 방지하는 데 중점을 둡니다.

### Phase 1: 핵심 기능 프로토타입 구축 (Foundation & Core UX)
**목표:** 게임 플레이 핵심 메카닉과 'Neon Vibe' 디자인 시스템을 통합하여, 사용자가 최소한의 인풋으로 재미를 느낄 수 있는 작동 가능한 프로토타입 완성.

| 마일스톤 | 세부 목표 | 주요 산출물 및 검증 항목 | 책임 에이전트 |
| :--- | :--- | :--- | :--- |
| **1.1 디자인 시스템 통합** | 'Neon Vibe' 디자인 명세($R_{Stability}$, Dynamic Effects)를 기반으로 UI 컴포넌트의 최종 시각적 일관성 확보. | `designer.md`에 명시된 레이아웃 및 아트 에셋이 실제 웹 페이지에 정확히 매핑되는지 확인. | Designer |
| **1.2 게임 로직 구현** | 핵심 게임 메카닉(예: 네온 서바이버 요소)을 독립적인 환경에서 안정적으로 작동하도록 구현. | `chick-game-kit` 또는 `neon-survivor-kit`의 기본 동작이 오류 없이 실행되는지 확인 (단위 테스트). | 코다리 |
| **1.3 기초 UX 흐름** | 게임 시작, 레벨업, 보스전 등 핵심 상호작용에 대한 사용자 플로우 정의 및 구현. | 최소 기능 제품(MVP) 수준의 인터랙션이 명확한 피드백을 제공하는지 검증. | 코다리, Designer |

### Phase 2: 안정성 통합 및 수익화 연동 (Stability & Monetization Integration)
**목표:** 핵심 게임 플레이 위에 시스템 안정성($R_{Stability}$) 데이터 연동과 IAP(인앱 결제) 흐름을 안전하게 통합하여 비즈니스 모델을 구현.

| 마일스톤 | 세부 목표 | 주요 산출물 및 검증 항목 | 책임 에이전트 |
| :--- | :--- | :--- | :--- |
| **2.1 안정성 데이터 연동** | 시스템 안정성 지표($R_{Stability}$)를 실시간 또는 주기적으로 모니터링하고, 이 데이터를 게임 내 동적 효과에 연동하는 API/모듈 구현. | `developer.md`에서 정의된 $R_{Stability}$ 데이터가 웹 인터페이스에 시각적으로 반영되는지 **테스트 통과 확인**. | 코다리 |
| **2.2 IAP 흐름 통합** | 프리미엄 티어(Pro/VIP) 전환 및 결제 승인 과정이 보안적, 기술적으로 안전하게 구현됨을 보장. | PayPal 연동 모듈의 End-to-End 테스트 완료. 결제 성공/실패 시 시스템 안정성($R_{Stability}$) 로그 기록 확인. | 코다리 (Backend Focus) |
| **2.3 데이터 가드 레일 구축** | 외부 입력(API, 파일)에 대한 적절한 오류 처리 및 권한 검증 로직을 강화하여 잠재적 취약점 제거. | 모든 외부 통신 지점에 대한 예외 처리(`try-catch` 또는 Guard Clause)가 코드 전반에 걸쳐 적용되었는지 검토. | 코다리 |

### Phase 3: 최종 검증 및 출시 준비 (Final Validation & Launch Readiness)
**목표:** 완성된 시스템 전체에 대한 포괄적인 안정성 테스트를 수행하고, 마케팅 목표 달성을 위한 배포 환경을 완벽히 준비.

| 마일스톤 | 세부 목표 | 주요 산출물 및 검증 항목 | 책임 에이전트 |
| :--- | :--- | :--- | :--- |
| **3.1 통합 시스템 부하 테스트** | 동적 효과 연동 및 IAP 흐름이 최대 사용자 트래픽에서도 지연 없이 작동하는지 확인. | $R_{Stability}$ 데이터 처리 시뮬레이션을 통한 시스템 병목 현상 분석 및 해결 완료. | 코다리, Designer |
| **3.2 최종 안정성 검증** | 전체 시스템에 대한 **시스템 안정성 증명($T_{Stable}$)**을 최종적으로 확보. | 모든 핵심 기능(게임 플레이, 결제, 데이터 로깅)이 정의된 $R_{Stability}$ 기준을 충족함을 입증하는 보고서 및 로그 확보. | 코다리 |
| **3.3 배포 환경 준비** | 웹사이트의 성능 최적화(Lighthouse 점수 개선) 및 최종 마케팅 콘텐츠(영상, 캡션)와의 통합 검토. | 최종 버전 빌드 및 호스팅 환경 설정 완료. (마케팅 팀과의 싱크 확인). | 모든 에이전트 협업 |
