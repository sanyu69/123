### 📝 최종 시각화 및 시스템 가이드라인 산출물

다음은 `Neon_Vibe_Palette`와 아이콘 디자인 가이드라인을 기반으로 한 컴포넌트 시각화의 최종 명세입니다. 이 명세는 개발팀이 CSS와 SVG를 구현할 때 참고해야 할 모든 구체적인 값을 포함합니다.

#### 1. 브랜드 컬러 시스템: `Neon_Vibe_Palette` 확정 (CSS 변수)
| 변수명 | 색상 코드 (Hex) | 용도 | 설명 |
| :--- | :--- | :--- | :--- |
| `--color-primary` | `#00FFFF` | 핵심 강조 (Gauge Fill) | 밝은 시안/네온 블루. 시스템 안정성(`R_Stability`) 상태 표시. |
| `--color-secondary` | `#FF00FF` | 보조 강조 (Hover/Active) | 마젠타/핫핑크. 인터랙션 및 알림에 사용. |
| `--color-background` | `#0A0A1A` | 배경색 | 짙은 네이비/블랙 그라데이션 기반의 깊은 공간감 표현. |
| `--color-text` | `#FFFFFF` | 기본 텍스트 | 명확한 가독성을 위한 순백색. |
| `--color-stable` | `#00FF7F` | 안정 상태 (Success) | 네온 그린. $R_{Stability}$가 높을 때의 긍정적 신호. |
| `--color-warning` | `#FFFF00` | 경고 상태 (Warning) | 네온 옐로우. 주의 필요 시 사용. |
| `--color-danger` | `#FF4500` | 위험 상태 (Danger) | 오렌지 레드. 심각한 오류 또는 낮은 안정성 표시. |

#### 2. 아이콘 디자인 가이드라인 (SVG 스타일)
아이콘은 **네온 테크(Neon Tech)** 스타일을 기본으로 하며, 모든 요소는 `--color-primary` 또는 상태 색상(`--color-stable`, `--color-danger`)을 사용하여 발광하는 듯한 효과를 줍니다.

*   **스타일:** 라인 아트 기반의 기하학적 형태. 모서리는 날카롭고 각진 디자인 지향.
*   **아이콘 세트 예시 (Stability Gauge 관련):**
    *   **안정 상태 (Stable):** 원형 테두리 안에 중앙에 채워진 게이지 바. 색상은 `--color-stable`를 사용하며, 경계선은 `--color-primary`로 처리.
    *   **경고 상태 (Warning):** 동일한 구조이나 배경색이 `--color-warning`으로 변경되며, 테두리는 `--color-danger`의 약한 톤으로 강조.
    *   **위험 상태 (Danger):** 경고 상태보다 더 강렬하게 `--color-danger`를 사용하여 시각적 긴급성을 극대화.

#### 3. 컴포넌트 레이아웃 및 구조 명세 (Stability Gauge)

**목표:** $R_{Stability}$ 값을 시각적으로 표현하고, 동적 이펙트를 잠금(Lock) 상태로 설정하는 UI를 디자인합니다.

*   **컨테이너:** 전체 게이지 영역은 `--color-background`(`\#0A0A1A`)의 깊은 배경 위에 투명도를 가진 네온 오버레이가 적용된 형태로 구현하여, 3D적인 '네온 아우라' 효과를 부여합니다.
*   **게이지 바 (Gauge Bar):**
    *   **기본 상태:** 게이지 바는 `--color-primary`로 채워지며, 선형으로 표시됩니다.
    *   **동적 이펙트 잠금 (Lock State):** 게이지 바 전체가 불투명한 네온 레이어로 덮이며, 중앙에 **'LOCK'** 텍스트가 `--color-secondary`로 발광합니다. 이 상태에서는 애니메이션이 정지됩니다.
    *   **$R_{Stability}$ 값 표시:** 중앙에는 현재 $R_{Stability}$ 수치(예: 85%)를 `--color-text`로 매우 크고 굵게 표시하여 즉각적인 정보 전달을 최우선합니다.

#### 4. 개발팀 전달 요약 (Action Items for Dev Team)
개발팀은 다음의 명세를 기반으로 `stability_gauge.html` 및 관련 SVG/CSS를 구현해야 합니다.

1.  **CSS 변수 적용:** 위 표에 정의된 모든 `--color-*` 변수를 프로젝트 전역에 적용합니다.
2.  **Gauge Bar 구현:** 게이지 바는 CSS Gradient 또는 `conic-gradient`를 활용하여 네온 효과를 시뮬레이션하며, 상태별 색상(`--color-stable`, `--color-danger`)을 동적으로 변경하도록 구현합니다.
3.  **Lock State UI:** '동적 이펙트 잠금' 상태일 때 게이지 바 위에 오버레이 레이어를 적용하고 텍스트를 중앙 정렬하여 시각적 고정 효과를 구현합니다.
4.  **Icon Integration:** 모든 아이콘(예: 안정성 아이콘)은 정의된 네온 스타일 SVG로 제작하여, 색상 변화에 따라 발광하는 듯한 느낌을 부여합니다.
