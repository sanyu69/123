## 💡 최종 비주얼 가이드라인 요약 (Stability-based Premium Neon)

**핵심 목표:** 시스템 안정성($R_{Stability}$)과 프리미엄 경험을 강조하는 시각적 일관성 확보.
**스타일:** 네온(Neon) 미학 + 안정감 있는 구조.

### 1. 컬러 팔레트: `Neon_Vibe_Palette` 확정
| 역할 | 색상 이름 | HEX 코드 | 용도 및 의미 |
| :--- | :--- | :--- | :--- |
| **Primary Accent** (안정성 강조) | Deep Stability Blue | `#007BFF` | 시스템 안정성, 신뢰, 메인 액션 버튼 |
| **Secondary Accent** (네온 효과) | Electric Cyan | `#00FFFF` | 네온/글로우 효과, 프리미엄 요소 강조 |
| **Background Base** | Dark Slate | `#1A1A2E` | 어두운 배경으로 네온 효과 극대화 |
| **Highlight/Warning** (안정성 지표) | Stability Green | `#32CD32` | 성공, 안정성 게이지, 긍정적 피드백 |

### 2. 타이포그래피 가이드라인
*   **제목/헤드라인:** `Orbitron` 또는 유사한 산세리프 계열의 강렬하고 미래적인 폰트 (볼드체).
*   **본문/설명:** `Space Mono` 또는 `Roboto Mono`와 같이 기술적이고 구조적인 느낌을 주는 모노스페이스 폰트.

## 🖼️ 핵심 비주얼 에셋 컨셉 3안 (썸네일 & 게시물용)

다음은 유튜브 썸네일과 인스타그램 포스트에 바로 적용 가능한 3가지 디자인 컨셉입니다. 모든 레이아웃은 **Dark Slate** 배경 위에 네온 색상을 강조합니다.

### 컨셉 1: Stability Gauge 강조형 (가장 추천)
*   **목표:** 시스템 안정성($R_{Stability}$)을 가장 직관적으로 시각화하여 신뢰도를 극대화.
*   **주요 요소:** 중앙에 크고 선명한 **Stability Gauge** 컴포넌트를 배치하고, 주변에 네온 효과를 적용합니다.
*   **색상 사용:** `Deep Stability Blue` (테두리/배경)와 `Stability Green` (게이지 채움 부분)을 주력으로 사용.
*   **타이포그래피:** 제목은 `Orbitron` (Electric Cyan), 안정성 수치(%)는 매우 크게 강조.

| 항목 | 상세 명세 |
| :--- | :--- |
| **배경** | `#1A1A2E` (Dark Slate) |
| **핵심 시각 요소** | 중앙에 원형/반원 형태의 Stability Gauge 컴포넌트 배치. 게이지 채움 부분은 `#32CD32`. |
| **텍스트 레이아웃** | 상단: 핵심 문구 (Orbitron, Electric Cyan). 하단: $R_{Stability}$ 수치 (매우 크게, 화이트 또는 Cyan). |
| **톤** | 기술적이고 신뢰감 넘침. |

### 컨셉 2: Neon Vibe & Premium Focus형
*   **목표:** 프리미엄(Pro/VIP) 기능의 매력과 네온 미학을 강조하여 시각적 흥미 유발.
*   **주요 요소:** 제품 UI 목업 또는 핵심 아트 에셋을 중앙에 배치하고, 주변에 역동적인 네온 블러 효과를 추가합니다.
*   **색상 사용:** `Electric Cyan`과 `Deep Stability Blue`를 대비시켜 사용하며, 전체적으로 글로우(Glow) 효과 극대화.
*   **타이포그래피:** 제목은 대담한 산세리프체로 강조.

| 항목 | 상세 명세 |
| :--- | :--- |
| **배경** | `#1A1A2E` (Dark Slate)에 은은한 그라데이션 적용. |
| **핵심 시각 요소** | Pro/VIP 기능의 UI 목업(예: 잠금 해제 화면)을 중앙에 배치. 테두리와 텍스트는 `Electric Cyan`. |
| **텍스트 레이아웃** | 제목은 크고 강조하며, 핵심 가치 문구(Stability-based Premium)를 작은 글씨로 하단에 배치. |
| **톤** | 세련되고 프리미엄함. |

### 컨셉 3: Data & Flow 강조형 (스크롤/릴스 최적화)
*   **목표:** 정보 전달과 과정(Flow)을 시각적으로 연결하여 흥미를 유발하고 클릭을 유도.
*   **주요 요소:** 단계별 프로세스를 화살표와 선으로 연결하여 동적인 흐름을 표현합니다.
*   **색상 사용:** `Deep Stability Blue`를 메인 라인으로, 중간에 `Electric Cyan`의 강조점을 삽입하여 데이터의 흐름을 보여줍니다.
*   **타이포그래피:** 간결하고 구조적인 모노스페이스 폰트를 사용하여 정보 전달력을 높임.

| 항목 | 상세 명세 |
| :--- | :--- |
| **배경** | `#1A1A2E` (Dark Slate) |
| **핵심 시각 요소** | 왼쪽에서 오른쪽으로 흐르는 데이터 플로우(Flow) 라인과 각 단계별 아이콘 배치. |
| **텍스트 레이아웃** | 각 단계의 제목은 간결하게 표시하고, 연결선에는 `Electric Cyan`을 사용하여 움직임을 강조. |
| **톤** | 정보 전달적이고 역동적임. |
