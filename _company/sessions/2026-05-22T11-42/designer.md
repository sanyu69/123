# 💎 시스템 안정성 기반 가격 플랜 비주얼 디자인 가이드라인

## 1. 디자인 컨셉: "Reliable Neon" (신뢰할 수 있는 네온)

**핵심 메시지:** 화려한 네온의 역동성을 유지하되, 각 티어는 **'흔들림 없는 견고함'**과 **'최고 수준의 안정성'**을 시각적으로 전달해야 합니다. 단순한 기능 차이가 아닌, 시스템 품질의 차이를 보여줍니다.

**시각적 톤앤매너:**
*   **스타일:** 하이테크, 미니멀리즘 기반의 네온 스타일. 복잡한 요소는 최소화하고, 안정성을 상징하는 구조적이고 깔끔한 레이아웃을 강조합니다.
*   **분위기:** 차갑고(Cool) 신뢰감 있으며(Trustworthy), 프리미엄적인(Premium) 느낌을 부여합니다.

## 2. 브랜드 컬러 시스템 (Color System: $R_{Stability}$ 반영)

'Neon\_Vibe\_Palette'를 기반으로 각 티어에 따라 안정성과 등급을 시각적으로 구분하는 색상 팔레트를 정의합니다.

| 요소 | 역할 | Hex Code | RGB | 설명 및 안정성 연관성 |
| :--- | :--- | :--- | :--- | :--- |
| **Primary (Base)** | 배경, 기본 텍스트 | `#1A1A2E` | (26, 26, 46) | 깊고 어두운 네이비/블랙. 안정감과 고급스러움을 제공하며 모든 요소의 기반이 됩니다. |
| **Accent (Neon)** | 핵심 강조 색상 | `#00FFFF` | (0, 255, 255) | 기본 네온 컬러. 역동성과 게임적인 느낌을 유지합니다. |
| **Stability Tier Color** | **Pro/VIP 강조** | `#00FF7F` | (0, 255, 127) | **안정성 색상.** 녹색 계열은 '안전', '완료', '성공'을 상징하여 안정성을 직관적으로 전달합니다. |
| **Base Tier Color** | Basic 강조 | `#AAAAAA` | (170, 170, 170) | 중립적인 회색. 기본 제공 및 시작점을 나타냅니다. |

## 3. 타이포그래피 시스템 (Typography System)

시스템의 명확성과 기술적 무결성을 강조하기 위해 기하학적이고 깔끔한 산세리프 폰트를 사용합니다.

*   **Headline (가격/티어명):** **'Orbitron'** 또는 유사한 기하학적 산세리프체. 강하고 미래지향적인 느낌을 주어 프리미엄 이미지를 강화합니다.
*   **Body Text (설명):** **'Inter'** 또는 **'Roboto'**. 가독성이 최우선이며, 복잡한 설명도 명확하게 전달하여 시스템의 투명성을 확보합니다.

## 4. 비주얼 에셋 방향성: 가격 플랜 목업 구성 (Mockup Layout)

목업은 세로형(Portrait) 레이아웃을 기본으로 하며, 수직적 흐름(Flow)을 통해 등급 상승을 표현합니다.

### A. 전체 구조 (Layout Flow)

1.  **상단 헤더:** "Stability-Focused Pricing" 또는 "System Reliability Tiers"와 같은 안정성 중심의 캐치프레이즈 배치.
2.  **티어 시각화:** 세 개의 카드(Basic, Pro, VIP)가 수직으로 명확하게 분리되어 배치됩니다. 각 카드는 경계선과 색상으로 구분됩니다.
3.  **핵심 가치 강조:** 각 티어의 가장 중요한 특징(예: Pro는 '자동 백업 활성화')을 아이콘과 함께 **Stability Tier Color (`#00FF7F`)**로 명확하게 표시합니다.

### B. 개별 티어 디자인 상세 (Tier Details)

| 티어 | 시각적 강조점 | 핵심 비주얼 요소 | 안정성 전달 방식 |
| :--- | :--- | :--- | :--- |
| **Basic** | 시작점, 접근성 | Neutral Gray 배경, 기본 네온 색상 사용. | '기본 제공'이라는 명확한 기준 제시. |
| **Pro (Focus)** | **최대 강조 영역** | **Stability Tier Color (`#00FF7F`)를 핵심 테두리나 아이콘에 적용.** 가장 두꺼운 경계선과 미묘하게 빛나는 효과(Glow) 적용. | "이것이 우리가 제공하는 안정성의 표준이다."라는 메시지 전달. |
| **VIP** | 최고 수준, 희소성 | 더 깊은 그림자(Depth)와 고급스러운 톤 적용. 골드/플래티넘 느낌의 미세한 하이라이트 추가. | "최고의 무결성과 지원을 보장한다."라는 프리미엄 경험 강조. |

## 5. 디자인 가이드 요약 (Design System Summary)

*   **핵심 색상:** `#1A1A2E` (Base), `#00FFFF` (Accent), **`#00FF7F` (Stability)**
*   **폰트 조합:** Orbitron (제목), Inter (본문)
*   **레이아웃 원칙:** 수직적 흐름을 통해 등급 상승을 표현하고, Pro/VIP에는 Stability Color를 사용하여 시각적 우위를 명확히 한다.
*   **에셋 스타일:** 모든 요소는 네온 효과(Soft Glow)와 기하학적인 선을 사용하여 하이테크 및 안정적인 느낌을 유지한다.
