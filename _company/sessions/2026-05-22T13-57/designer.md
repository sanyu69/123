## 🖼️ 유튜브 썸네일 컨셉 3안 (Stability Emphasis)

**핵심 디자인 원칙:**
*   **스타일:** Reliable Neon (강렬한 네온 효과, 깊이감 있는 배경)
*   **톤:** 시스템 안정성($R_{Stability}$) 강조 (신뢰감, 기술적 우위)
*   **컬러 팔레트:** 'Neon_Vibe_Palette' 준수 (주로 네온 블루/시안, 딥 블랙/다크 그레이 배경 사용)

### 컨셉 1: The Core Stability (시스템 안정성 극대화형)

**목표:** 시스템의 견고함과 신뢰성을 가장 직접적으로 강조하여 프리미엄 인식을 심어줍니다.
**핵심 메시지:** "흔들림 없는 안정성"
**주요 비주얼 요소:**
1.  **배경:** 딥 블랙 또는 다크 그레이 그라데이션 (깊이감 부여).
2.  **네온 효과:** 메인 타이틀과 핵심 숫자에 강한 **시안(Cyan) 네온**을 사용하여 기술적이고 미래지향적인 느낌 강조.
3.  **중앙 오브젝트:** 중앙에 'R_Stability' 또는 '100%'와 같은 안정성을 상징하는 기호나 그래프 라인을 네온으로 배치.
4.  **텍스트 배치:** 타이틀은 중앙 상단, 안정성 메시지는 하단에 명확하게 분리하여 배치.

| 항목 | 세부 내용 |
| :--- | :--- |
| **주요 색상** | 배경: `#0A0A14` (딥 블랙), 네온: `#00FFFF` (시안) |
| **레이아웃** | 텍스트는 중앙 정렬. 안정성 지표를 시각적 중심에 배치. |
| **폰트 스타일** | 산세리프 계열의 두꺼운 폰트로 기술적인 느낌 강조. |
| **CTA/키워드** | 메인 타이틀: `SYSTEM STABILITY` / 서브: `Reliable Neon` |

### 컨셉 2: The Neon Glow (미학적 매력 강조형)

**목표:** 'Neon Vibe'의 시각적 매력을 극대화하여 게임의 트렌디함과 몰입감을 어필합니다.
**핵심 메시지:** "트렌디한 경험"
**주요 비주얼 요소:**
1.  **배경:** 미세하게 움직이는 듯한 네온 그라데이션 (블루에서 핑크/퍼플 계열로 은은하게 전환).
2.  **네온 효과:** 전체적으로 부드럽고 화려한 **마젠타(Magenta) 및 퍼플(Purple)** 계열의 네온을 사용하여 시각적 매력을 극대화.
3.  **중앙 오브젝트:** 게임의 가장 매력적인 아트 에셋(예: 보스, 특수 효과)을 중앙에 배치하고 네온으로 하이라이트.
4.  **텍스트 배치:** 역동적인 대각선 구도를 사용하여 시선을 유도.

| 항목 | 세부 내용 |
| :--- | :--- |
| **주요 색상** | 배경: `#1A0A2C` (딥 퍼플), 네온: `#FF00FF` (마젠타/핑크) |
| **레이아웃** | 역동적인 대각선 구도. 텍스트를 가장자리나 비어있는 공간에 배치하여 'Glow' 효과 극대화. |
| **폰트 스타일** | 약간의 곡선미가 가미된 네온 스타일 폰트 사용. |
| **CTA/키워드** | 메인 타이틀: `NEON SURVIVAL` / 서브: `Ultimate Glow` |

### 컨셉 3: The Data Flow (정보 중심 및 전환 유도형)

**목표:** IAP(수익화) 전환에 필요한 '기능적 이점'을 명확히 제시하여 클릭과 구매를 유도합니다.
**핵심 메시지:** "차이를 경험하라"
**주요 비주얼 요소:**
1.  **배경:** 그리드 또는 데이터 흐름(Data Flow) 패턴의 미니멀한 배경.
2.  **네온 효과:** 명확하고 깨끗한 **밝은 블루 네온**을 사용하여 정보 전달의 명료함을 확보.
3.  **중앙 오브젝트:** Basic, Pro, VIP 티어 간의 차이를 상징하는 세 개의 분리된 박스 또는 아이콘을 시각적으로 대비시켜 배치.
4.  **텍스트 배치:** 핵심 이점(Benefit)과 가격 플랜(Tier)을 명확히 구분하여 제시.

| 항목 | 세부 내용 |
| :--- | :--- |
| **주요 색상** | 배경: `#111122` (어두운 블루 그레이), 네온: `#00BFFF` (밝은 블루) |
| **레이아웃** | 좌측에 혜택, 우측에 가격 플랜을 배치하는 분할 구도. |
| **폰트 스타일** | 매우 깔끔하고 가독성이 높은 산세리프체. 수치 강조. |
| **CTA/키워드** | 메인 타이틀: `UPGRADE NOW` / 서브: `Pro vs VIP` |
