# IAP 티어 비주얼 & 기능 분리 명세서 (Phase 1)

## 1. 디자인 목표 및 원칙

**목표:** Pro와 VIP Tier 간의 가치 차이($R_{Stability}$ 극대화)를 시각적으로 명확히 구분하여, VIP Tier 사용자가 최고 수준의 안정성과 독점적인 접근 권한을 즉시 인지하도록 한다.

**핵심 원칙:**
1.  **$R_{Stability}$ 반영:** VIP Tier는 Pro Tier보다 시각적 계층(Visual Hierarchy)과 미학적 디테일에서 명확하게 '안정성'과 '프리미엄'을 강조해야 한다.
2.  **Neon Vibe 유지:** 전체적인 디자인 톤앤매너('Neon\_Vibe\_Palette')는 유지하되, VIP Tier에서는 더 깊고 정제된(Refined) 네온 효과를 사용한다.
3.  **명확한 차별화:** 기능적 우위가 시각적으로 직관적으로 연결되어야 한다.

## 2. 비주얼 시스템 (컬러 및 타이포차)

| 요소 | Pro Tier (9,900원/월) | VIP Tier (19,900원/월) | 디자인 의도 |
| :--- | :--- | :--- | :--- |
| **Primary Accent Color** | `#00FFFF` (밝은 시안/네온 블루) | `#00BFFF` (더 깊고 안정적인 아쿠아 블루) + 미묘한 골드(Accent) | VIP Tier에 고급스러움과 안정감을 부여하여 가격 프리미엄을 시각화. |
| **Background Tone** | Dark Mode Base (`#1A1A2E`) | Deeper Space Blue (`#0A0D14`) | VIP Tier에 더 깊은 공간감과 몰입도를 제공하여 '최고 수준의 안정성'을 암시. |
| **Border/Shadow Style** | Sharp, Clean Glow | Soft, Diffused Depth (Subtle Layering) | Pro는 기능적 명료성을 강조, VIP는 심층적인 레이어링과 고급스러움을 강조. |
| **Typography Weight** | Medium-Bold | Semi-Bold + Light Emphasis | 정보 전달의 명확성(Pro)과 미학적 세련됨(VIP)을 분리하여 표현. |

## 3. 기능적 시각화 및 레이아웃 차별점 (UI/UX Flow)

### A. 대시보드 및 상태 표시 (Stability Focus)

| 기능 영역 | Pro Tier 구현 방식 | VIP Tier 구현 방식 |
| :--- | :--- | :--- |
| **시스템 안정성 표시** | 실시간 서버 상태 표시 (온도, 메모리 사용량 등) | **심층 분석 대시보드:** $R_{Stability}$의 세부 지표(Latency, Error Rate 추이, 서버 부하 예측 모델)를 시각화하여 제공. |
| **데이터 시각화** | 기본 통계 그래프 (실시간/과거 추이) | **맞춤형 보고서 생성:** 사용자가 정의한 KPI에 기반한 예측 분석 차트 및 맞춤형 리포트 자동 생성 기능 제공. |
| **광고 제거** | UI 요소에 광고 배너가 완전히 제거됨. | **프리미엄 인터페이스 레이어:** 모든 기본 UI 요소에 'VIP Lock' 상태를 표시하며, 광고 없이 독점적인 인터페이스가 활성화됨을 명시. |

### B. 인앱 결제(IAP) 흐름 차별화

1.  **Tier 선택 화면:** Pro와 VIP 옵션은 단순한 가격 비교가 아닌, **"안정성 수준 비교"** 프레임으로 제시한다.
    *   **Pro:** "표준 안정성 확보 (기능 확장)"
    *   **VIP:** "**최고 안정성 보장** (독점 분석 및 무결성 보장)"

2.  **업그레이드 시점:** VIP로 업그레이드 시, 단순히 기능 해제가 아니라, **"시스템 $R_{Stability}$ 레벨 상향"** 메시지를 중앙에 배치하여 가치 전환을 명시한다.
