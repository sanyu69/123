## VIP 안정성 대시보드 UI/UX 설계 제안

### 1. 디자인 목표 (Design Goals)
*   **핵심 목표:** $S_{VIP}$ 지표를 통해 사용자에게 **'절대적인 시스템 안정성'에 대한 신뢰감**을 시각적으로 전달하고, VIP 구독의 가치를 명확히 입증한다.
*   **톤 앤 매너:** 기존의 'Neon Vibe' 미학을 유지하되, 불안정성(Risk)이 낮고 안정성(Assurance)이 극대화된 **'Deep Stability'** 또는 **'Secure Neon'** 스타일로 전환한다. (차가운 블루/시안 계열에 골드/앰버 포인트 사용)
*   **사용자 경험:** 복잡한 기술 지표를 직관적이고 즉각적으로 이해할 수 있도록 단순화하며, 불안정 시점에는 명확한 경고와 해결책을 제시한다.

### 2. 핵심 시각화 요소 및 데이터 매핑

| 데이터 항목 | 시각화 방식 | 디자인 의도 |
| :--- | :--- | :--- |
| **$S_{VIP}$ 지표 (핵심)** | 중앙 집중형 대형 게이지 또는 원형 진행률 표시 | 현재 안정성 수준을 한눈에 파악하게 하며, VIP 등급의 핵심 가치임을 강조. |
| **안정성 추이 (History)** | 꺾은선 그래프 (Line Chart) + 히트맵 배경 | 시간 경과에 따른 $S_{VIP}$ 변화 추이를 보여주며, 안정성의 일관성을 시각화. 불안정 구간은 명확히 표시. |
| **시스템 상태 (Real-time)** | 상태 표시등 (Status Indicator) 및 색상 코드 | 실시간으로 시스템의 현재 상태(Online/Stable/Warning)를 직관적으로 전달. |
| **차별점 비교** | 타임라인 또는 스케일 비교 바 (Progress Bar) | Basic, Pro, VIP 티어 간 $S_{VIP}$ 점수의 차이를 시각적으로 비교하여 업그레이드의 명확한 이점을 제시. |

### 3. 레이아웃 및 컴포넌트 상세 설계 (Mockup Concept)

#### A. 대시보드 상단: 안정성 요약 섹션 (Stability Summary Header)
*   **구성:** 가장 크게 $S_{VIP}$ 점수와 현재 상태를 표시한다.
    *   **$S_{VIP}$ Score:** 중앙에 크고 선명하게 표시 (예: 98.5/100). 색상은 안정성에 따라 녹색(Green)에서 황금색(Gold)으로 변화.
    *   **Status Tag:** 현재 상태를 명시하는 태그 (예: **[STABLE]** 또는 **[CRITICAL WARNING]**).
    *   **VIP Promise:** "VIP 보장 안정성 레벨"이라는 문구를 함께 배치하여 프리미엄 가치를 강조.

#### B. 중앙 영역: 추이 및 상세 분석 (Trend & Detail Area)
*   **시각화:** $S_{VIP}$의 7일/30일 추이 꺾은선 그래프를 메인으로 배치한다.
*   **보조 시각화:** 배경에 미세한 그리드 패턴을 적용하여 기술적이고 정교한 느낌을 부여한다 (Neon Vibe 유지).
*   **상세 분석 패널:** 아래쪽에 $S_{VIP}$ 점수가 하락/상승한 주요 원인(예: Latency Spike, DB Load)을 텍스트와 작은 아이콘으로 요약하여 표시한다.

#### C. 좌측/우측 사이드바: 기능 및 설정 (Navigation & Configuration)
*   **기능 분류:** '안정성 대시보드', 'API 상태', '과거 로그', 'VIP 관리' 등으로 명확하게 분리한다.
*   **상호작용:** 사용자가 특정 지표를 클릭하면 해당 데이터가 중앙 영역에 확대되어 표시되는 인터랙티브한 디자인을 채택한다.

### 4. 색상 팔레트 제안 (Color Palette Proposal)

| 용도 | 색상 코드 (HEX) | 역할 및 의도 |
| :--- | :--- | :--- |
| **Primary (Stability)** | `#00AEEF` (Bright Cyan/Blue) | 시스템의 활성 상태 및 기본 안정성. 네온 느낌 유지. |
| **Success (Stable)** | `#39FF14` (Neon Green) | $S_{VIP}$가 매우 높을 때, 최고 수준의 신뢰를 표현. |
| **Warning (Risk)** | `#FFC72C` (Amber/Gold) | 경고 상태. 주의를 요구하며 즉각적인 조치가 필요함을 알림. |
| **Critical (Failure)** | `#FF3B30` (Bright Red) | 심각한 불안정성 발생 시, 긴급 대응을 촉구하는 색상. |
| **Background** | `#0A0F14` (Deep Dark Blue/Black) | 네온 효과를 극대화하고 데이터가 돋보이게 하는 깊은 배경색. |
