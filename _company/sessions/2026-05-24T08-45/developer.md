CEO 지시사항을 확인했습니다. P1 우선순위인 **시스템 안정성($R_{Stability}$) 확보를 위한 핵심 모니터링 로직 구현 계획 수립 및 개발 착수 준비**에 집중하겠습니다.

이 작업은 단순한 기능 구현이 아니라, 기존의 $R_{Stability}$ 데이터 파이프라인과 API 연동의 무결성을 보장하는 것을 최우선 목표로 설정하고 진행할 것입니다.

현재까지 확보된 지식($R_{Stability}$ 관련 검증 기록)을 기반으로, 핵심 모니터링 로직 구현 계획(Roadmap)을 구체화하여 제시하겠습니다.

### 🚀 $R_{Stability}$ 핵심 모니터링 로직 구현 계획 (MVP 준비 단계)

**목표:** 시스템 안정성 지표($R_{Stability}$)를 실시간으로 정확하게 수집하고, 이를 기반으로 VIP/Pro 티어 전환에 필요한 데이터 무결성을 확보하는 모니터링 로직을 구현한다.

#### 1단계: 요구사항 및 데이터 흐름 정의 (Analysis & Specification)
**목표:** 모니터링 대상 지표와 API 연동 구조를 명확히 정의하여 개발 착수 전에 모든 이해관계자의 합의를 이끌어낸다.

1.  **모니터링 지표 확정 ($R_{Stability}$ Scope Definition):**
    *   $R_{Stability}$에 포함되어야 할 핵심 지표(예: 서버 응답 시간, 트랜잭션 성공률, 결제 시스템 연동 안정성 등)를 명확히 정의합니다. (기존 설계 검토 필요)
2.  **데이터 수집 경로 매핑:**
    *   실시간 데이터($R_{Stability}$ 지표)가 시스템에서 생성되어 모니터링 레이어에 도달하는 전체 데이터 흐름(Source $\rightarrow$ Pipeline $\rightarrow$ API Endpoint)을 시각화하고 명세합니다. (이전 작업 기반 검증)
3.  **API 인터페이스 설계:**
    *   모니터링된 $R_{Stability}$ 데이터를 외부 시스템(프론트엔드/결제 시스템)에 안전하게 전달하기 위한 최소한의 API 엔드포인트(`stability`, `vip-status` 관련) 명세서를 확정합니다. (이전 작업 기반 검증)

#### 2단계: 핵심 로직 개발 및 통합 (Implementation & Integration)
**목표:** 1단계에서 정의된 스펙에 따라 $R_{Stability}$ 데이터를 수집, 검증하고 API로 노출하는 백엔드 모듈을 구현한다.

1.  **실시간 데이터 수집 모듈 개발 ($\text{Monitor Module}$):**
    *   `stability_monitor.py`를 확장하여, 실제 시스템 상태를 반영하는 지표를 안정적으로 수집하도록 로직을 개선합니다. (특히 파일 시스템 접근 및 권한 문제를 재확인하며 구현)
2.  **데이터 무결성 검증 로직 ($\text{Validation Layer}$):**
    *   수집된 $R_{Stability}$ 데이터가 허용 가능한 범위 내에 있는지, API 연동 시 발생하는 인증 오류(HTTP 401 문제 등)를 사전에 방지하는 가드(Guard) 로직을 추가합니다. (인증 안정성 확보)
3.  **API 구현:**
    *   정의된 명세에 따라 $R_{Stability}$ 및 VIP 상태 데이터를 제공하는 RESTful API 엔드포인트(`stability`, `vip-status`)를 구현하고 테스트합니다.

#### 3단계: 자기 검증 및 안정성 확보 (Verification & Hardening)
**목표:** 개발된 모듈이 예상대로 작동하며, 시스템의 안정성을 최종적으로 보장한다.

1.  **단위/통합 테스트 실행:**
    *   모든 데이터 수집 및 API 응답 과정에 대한 통합 테스트를 수행합니다. 특히 비정상적인 입력(Error Injection) 상황에서 시스템이 안전하게 처리하는지 검증합니다. ($\text{lint\_test}$ 또는 사용자 정의 테스트 루프 사용)
2.  **안정성 시각화 확인:**
    *   $R_{Stability}$ 지표가 대시보드에 반영될 때 데이터의 정확성이 시각적으로 보장되는지 최종 점검합니다.
