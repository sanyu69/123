# 🖼️ VIP/Pro 대시보드 비주얼 스펙 최종본 (마케팅용)

## 📌 디자인 철학 요약
**스타일:** 네온 미학(Neon Vibe) + 글래스모피즘(Glassmorphism)
**컬러 팔레트:** `#0A0A1A` (BG), `#00FFFF` (Cyan, Primary Glow), `#39FF14` (Neon Green, Secondary Glow).

## 🎯 핵심 비주얼 소스 요구사항

### A. Reels 1: 불안정성 $\rightarrow$ 안정성 전환 시퀀스
*   **목표:** 대비 효과 극대화.
*   **구성:** 좌측(불안정, Red/Gray Noise)과 우측(안정, Cyan/Green Glow).
*   **요소:** $R_{Stability}$ 게이지의 극적인 빛 변화에 집중할 것.

### B. Reels 2: 프리미엄 기능 강조 (대시보드 클로즈업)
*   **목표:** 시스템 안정성을 시각적 고급스러움으로 연결.
*   **구성:** 네온 테마의 깔끔한 대시보드 전체 화면.
*   **요소:** $R_{Stability}$ 지표가 중앙에 최대 밝기로 발광하며, 모든 데이터는 Cyan/Green 라인으로만 표현하여 미니멀리즘 유지.

### C. Instagram: 감성적 미학 (Aesthetic Post)
*   **목표:** 네온 분위기의 고해상도 이미지 제공.
*   **구성:** 깊은 어둠 속에서 Cyan과 Green의 광원이 스며드는 분위기 연출.
*   **요소:** 시각적 품질(HDR, 대비)을 최우선으로 하여 '프리미엄' 느낌을 강조할 것.

## ⚙️ 기술 명세 (개발 참고용)
*   **Primary Color Code:** `#00FFFF` / `#39FF14`
*   **Background Color Code:** `#0A0A1A`
*   **Font Suggestion:** Orbitron, Space Mono
