## 💎 VIP/Pro 대시보드 비주얼 시스템 명세 (Neon Stability 기반)

### 1. 핵심 미학 및 컬러 팔레트 (Neon Stability)

| 요소 | 설명 | 코드/값 | 적용 목표 |
| :--- | :--- | :--- | :--- |
| **주요 배경** | 깊이감 표현을 위한 어두운 네온 블랙 | `#0A0A14` (거의 검정, 미세한 보라 톤) | 깊이감(Depth) 확보 |
| **VIP 강조색** | 안정성과 프리미엄감을 표현하는 차가운 시안/블루 | `#00FFFF` (네온 시안) | VIP 티어의 '깊이' 강조 |
| **Pro 강조색** | 정보 밀도와 기술적 깊이를 표현하는 밝은 마젠타 | `#FF00FF` (네온 마젠타) | Pro 티어의 '밀도' 강조 |
| **보조 텍스트** | 안정적인 정보를 위한 은은한 회색 | `#B0B0C0` | 정보 구별 및 안정성 제공 |

### 2. VIP 대시보드 디자인 명세 (Depth Focus)

VIP 티어는 '깊이감(Depth)'을 통해 프리미엄 경험을 시각화합니다.

*   **레이아웃:** 깊은 그림자(Ambient Glow) 효과를 극대화하여 정보가 공간 속에 떠 있는 듯한 느낌을 연출합니다.
*   **요소 처리:** 모든 핵심 데이터 블록은 배경보다 한 단계 뒤에 위치하며, **`#0A0A14`** 배경 위에 **`#00FFFF`**의 은은한 광선(Glow)이 가장자리에서 미세하게 발산됩니다.
*   **차별점:** 정보 간의 거리를 넓게 설정하여 시각적 여유를 부여합니다.
*   **핵심 에셋 명세 (예시):**
    *   **VIP Status Bar:** 전체 화면의 10% 높이를 차지하며, `Y`축 깊이감을 표현하는 미세한 그라데이션 적용.
    *   **$R_{Stability}$ Gauge:** 시안 색상(`#00FFFF`)으로 채워지며, 바늘은 중앙에서 약간 아래로 위치하여 안정적인 상태를 암시.

### 3. Pro 대시보드 디자인 명세 (Density Focus)

Pro 티어는 '정보 밀도(Density)'를 통해 기술적 깊이와 효율성을 시각화합니다.

*   **레이아웃:** 정보 블록 간의 간격을 최소화하여 데이터를 압축적으로 표현하며, 전체 화면을 데이터로 채웁니다.
*   **요소 처리:** `Pro 강조색`인 **`#FF00FF`**를 사용하여 데이터 시각화 요소(차트, 지표)에 적용합니다.
*   **차별점:** 데이터 포인트가 촘촘하게 배열되어 높은 정보 밀도를 제공하며, 배경의 네온 효과는 최소화하여 데이터 자체에 집중하게 만듭니다.
*   **핵심 에셋 명세 (예시):**
    *   **Pro Data Grid:** 격자선(Grid Line)을 `FF00FF`로 설정하고, 데이터 셀은 `B0B0C0` 텍스트와 대비되도록 처리하여 정보의 밀도를 높입니다.
    *   **$R_{Stability}$ Indicator:** 마젠타 색상(`#FF00FF`)으로 채워지며, 지표가 높을수록 더 강렬한 네온 효과를 발산하도록 설정합니다.

### 4. 최종 디자인 가이드라인 요약 (개발자 전달용)

| 구분 | VIP (Depth) | Pro (Density) |
| :--- | :--- | :--- |
| **핵심 미학** | 깊이감 (Depth) | 정보 밀도 (Density) |
| **주요 색상** | `#00FFFF` (시안) | `#FF00FF` (마젠타) |
| **배경 처리** | 낮은 채도, 광선 효과 극대화 | 높은 대비, 선명한 데이터 표현 |
| **UI Flow 목표** | 안정적인 프리미엄 경험 제공 | 효율적이고 심층적인 분석 환경 제공 |
