## 💎 핵심 썸네일 Mockup 디자인 결과 (3안)

각 썸네일은 **'깊이(Depth)'**, **'밀도(Density)'**, **'대비(Contrast)'** 세 가지 컨셉을 기반으로 제작되었으며, 실제 Figma에 적용할 수 있는 구체적인 시각 명세로 제시합니다.

### 1. 컨셉 A: 깊이 강조 (Depth Focus)
*   **핵심 가치:** 몰입감과 차원 이동 (Depth)
*   **시각 목표:** 3차원적 공간감을 극대화하여 사용자가 클릭하지 않고도 게임의 심연으로 빨려 들어가는 느낌을 제공합니다.

| 요소 | 상세 디자인 명세 | 색상 코드 및 적용 방식 |
| :--- | :--- | :--- |
| **배경** | 깊은 어둠 또는 우주를 연상시키는 그라데이션 배경. 미묘한 3D 입체감이 느껴져야 함. | 배경: `#1A0233` (Deep Violet 계열)에서 `#0A051A` (거의 검은색)로 부드럽게 전환되는 **깊은 그라데이션**. |
| **텍스트 (Hook)** | 중앙에 배치하며, 입체감을 위해 그림자 효과를 강조. | 폰트: `Montserrat Bold` 또는 유사한 기하학적 산세리프체. 색상: `#00FF7F` (Neon Green). **강한 외곽선(Stroke) 적용**. |
| **레이아웃** | 중앙 정렬, 텍스트 뒤에 미묘한 광원 효과를 주어 깊이감을 연출. | 텍스트 주변에 Neon Green의 희미한 후광(Glow)을 적용하여 시각적 깊이를 더함. |
| **추가 요소** | 보스나 핵심 오브젝트의 실루엣을 배경에 은은하게 배치하여 '깊이'를 암시. |

### 2. 컨셉 B: 밀도 강조 (Density Focus)
*   **핵심 가치:** 압축된 경험과 최대 효율 (Density)
*   **시각 목표:** 정보와 에너지가 극도로 압축되어 폭발하는 듯한 높은 에너지 밀도를 표현하여, '가성비'가 아닌 '최고의 집중도'를 강조합니다.

| 요소 | 상세 디자인 명세 | 색상 코드 및 적용 방식 |
| :--- | :--- | :--- |
| **배경** | 단색 또는 매우 미세한 노이즈 패턴을 사용하여 시각적 혼란(Density)과 에너지를 표현. | 배경: `#0A1A0A` (짙은 녹색 계열). 텍스처를 아주 미세하게 추가하여 밀도를 높임. |
| **텍스트 (Hook)** | 텍스트 블록을 화면의 대부분을 채우며 배치. 정보의 밀도가 높음을 시각적으로 전달. | 폰트: `Bebas Neue` 또는 유사한 강력하고 압축적인 산세리프체. 색상: `#7FFF00` (밝은 Neon Green). **꽉 찬 블록 형태**. |
| **레이아웃** | 정보 위주로 배치. 핵심 가치(Depth/Density)를 숫자나 아이콘으로 밀도 높게 표시. | 텍스트 간의 여백을 최소화하고, 모든 요소를 화면에 최대한 밀집시켜 '압축된 경험'을 강조. |
| **추가 요소** | 데이터 라인이나 그래프 형태의 시각적 요소를 텍스트 주변에 삽입하여 정보 밀도를 표현. |

### 3. 컨셉 C: 대비 강조 (Contrast Focus)
*   **핵심 가치:** 명확한 차별화와 선택의 강요 (Contrast)
*   **시각 목표:** Deep Violet과 Neon Green의 극단적인 색상 대비를 통해 시선을 즉각적으로 사로잡고, VIP/Pro 전환이 필수적임을 강력하게 어필합니다.

| 요소 | 상세 디자인 명세 | 색상 코드 및 적용 방식 |
| :--- | :--- | :--- |
| **배경** | 좌우 또는 중앙을 극단적으로 분할하여 두 가지 영역의 대비를 명확히 제시. | 배경: 왼쪽 50%는 Deep Violet (`#6A0DAD`), 오른쪽 50%는 Neon Green (`#39FF14`)으로 **날카롭게 분리**. |
| **텍스트 (Hook)** | 중앙에 배치하여 두 색상의 경계를 관통하도록 디자인. | 폰트: `Impact` 또는 매우 두꺼운 산세리프체. 색상: 텍스트는 배경색과 대비되는 단일 색(예: 흰색 또는 밝은 회색)을 사용하되, **경계선에만 네온 컬러를 사용**. |
| **레이아웃** | 대각선 분할 또는 강한 수직 분할. 두 영역이 충돌하며 시각적 긴장감을 유발. | 대비되는 색상이 서로를 밀어내는 듯한 역동적인 레이아웃 구성. |
| **추가 요소** | 전환 버튼(CTA)을 Neon Green 영역에 배치하여 클릭 유도. |
