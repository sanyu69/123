### 📊 IAP 전환율 시뮬레이션 모델 입력 포맷 및 변수 목록

이 시뮬레이션 모델은 **'미학적 차별화(Cyan/Magenta)'**가 실제 **'수익화 안정성($S_{Success}$)'**에 미치는 영향을 예측하는 데 중점을 둡니다.

#### 1. 핵심 데이터 입력 포맷 (Data Input Format)
모든 데이터는 다음의 구조를 따르도록 준비해야 합니다.

| 필드명 | 설명 | 데이터 타입 | 필수 여부 | 비고 |
| :--- | :--- | :--- | :--- | :--- |
| **Tier** | 티어 구분 (Basic, Pro, VIP) | String | 필수 | - |
| **Price** | 해당 티어의 실제 가격 | Float | 필수 | USD 기준 |
| **Visual\_Factor** | 사용된 시각적 차별화 요소 | String/Numeric | 필수 | Cyan/Magenta 적용 여부 또는 강도 (1~5점) |
| **Conversion\_Data** | 해당 티어에서 발생한 실제 전환 데이터 | Float | 필수 | $S_{Success}$ 값 (예: 15%) |
| **Traffic\_Source** | 트래픽 유입 경로 | String | 선택 | (예: Organic, Paid, Social) |

#### 2. 필요한 주요 변수 목록 (Key Variables for Simulation)

시뮬레이션의 정확도를 높이기 위해 다음 핵심 변수들을 확보해야 합니다.

*   **가격 변수 ($\text{P}$):**
    *   $\text{P}_{\text{Basic}}$: Basic 티어 가격 ($4.99$)
    *   $\text{P}_{\text{Pro}}$: Pro 티어 가격 ($9.99$)
    *   $\text{P}_{\text{VIP}}$: VIP 티어 가격 ($19.99$)

*   **차별화 변수 ($\text{V}$):**
    *   $\text{V}_{\text{Cyan\_Effect}}$: Cyan 비주얼 적용에 따른 전환율 변화 계수 (기반값 대비 상승/하락 비율)
    *   $\text{V}_{\text{Magenta\_Effect}}$: Magenta 비주얼 적용에 따른 전환율 변화 계수

*   **성능 변수 ($\text{S}$):**
    *   $S_{\text{Base}}$: 시각적 차별화가 없을 때의 기본 성공률 (Baseline Success Rate)
    *   $R_{\text{Stability}}$: 시스템 안정성 지표 ($200\text{ms}$ 이하 유지 여부)

*   **결과 변수 ($\text{C}$):**
    *   $\text{C}_{\text{Predicted}}$: 시뮬레이션으로 예측된 최종 전환율.
