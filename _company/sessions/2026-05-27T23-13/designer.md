### ✨ 시각 시스템 기반 마케팅 소재 (The System Gauge & $R_{Stability}$ 강조)

**기반 원칙:**
*   **미학:** 'The System Gauge'의 네온/다이내믹 효과를 유지하며, 기술적 안정성($R_{Stability}$)을 신뢰와 성능으로 치환하여 시각화합니다.
*   **컬러 팔레트:** 기존 정의된 `Neon_Vibe_Palette` (예: 딥 블루, 네온 핑크/시안, 다크 그레이 배경)를 중심으로 사용합니다.
*   **핵심 메시지:** "단순한 게임이 아닌, 검증된 시스템을 경험하라."

---

### 🖼️ 소재 1안: The Stability Meter (웹사이트 핵심 배너 최적화)

**목표:** 웹사이트 랜딩 페이지 상단에 배치하여 즉각적인 신뢰감을 제공하고 클릭 유도.
**형식:** 가로형 배너 (Landscape)

| 항목 | 상세 내용 | 좌표/규격 |
| :--- | :--- | :--- |
| **배경** | 짙은 차콜 그레이 (#1a1a2e). 미세한 그리드 패턴 오버레이. | 1920x600px |
| **핵심 요소** | 중앙에 'The System Gauge'의 메인 게이지 시각화. $R_{Stability}$ 수치(예: 98.7%)가 가장 밝은 네온 블루로 표시됨. | 중앙 60% 영역 |
| **타이포그래피** | 제목: `R_STABILITY` (굵고 각진 산세리프, 네온 시안색 #00FFFF). 부제: '검증된 안정성' (얇은 고딕체, 화이트). | 제목 중앙 상단 |
| **액션 버튼** | "시스템 성능 확인하기" (네온 핑크 배경에 흰색 텍스트) | 우측 하단 배치 |
| **스타일 노트** | 게이지 바는 미세하게 깜박이며(Glow Effect), $R_{Stability}$ 수치가 높아질수록 선이 더욱 강렬하게 빛나도록 디자인. 전체적으로 차가우면서도 미래지향적인 느낌 강조. |

### 📱 소재 2안: Dynamic Flow (Instagram/Facebook 피드용)

**목표:** SNS 피드에서 시선을 사로잡고, 시스템의 역동성과 끊임없는 안정성을 감성적으로 전달.
**형식:** 세로형 (Portrait) - 9:16 비율

| 항목 | 상세 내용 | 좌표/규격 |
| :--- | :--- | :--- |
| **배경** | 다크 모드 배경 (#0a0a1a). 화면 중앙에서 위로 상승하는 에너지 라인(Flow Line) 효과. | 1080x1920px |
| **핵심 요소** | 'The Dynamic Flow'를 표현하는 빛의 흐름. 이 흐름이 $R_{Stability}$ 지표와 연결되어 끊임없이 움직이는 이미지를 사용. | 화면 전체 |
| **타이포그래피** | 중앙에 크게 `STABILITY IS FLOW` (네온 핑크/시안 그라데이션). 하단에 작은 글씨로 "$R_{Stability}$: 실시간 검증" 명시. | 중앙 배치 |
| **감성 강조** | 움직임과 안정성의 대비를 통해, '불안정함 속에서 발견한 완벽한 흐름'이라는 메시지 전달. 시각적 에너지 극대화. |

### 📊 소재 3안: The Trust Seal (마케팅 광고/이메일용)

**목표:** 잠재 고객에게 기술적 안정성이 곧 프리미엄 가치임을 확신시키고 신뢰를 구축.
**형식:** 정사각형 또는 직사각형 (Square/Banner)

| 항목 | 상세 내용 | 좌표/규격 |
| :--- | :--- | :--- |
| **배경** | 미니멀한 블랙 배경 (#000000). | 1200x800px |
| **핵심 요소** | 중앙에 원형의 '안정성 인증 마크' (Trust Seal) 배치. 이 마크 내부에는 $R_{Stability}$ 수치와 함께 "Verified System" 문구를 삽입. | 중앙 정렬 |
| **디테일** | 마크 주변을 얇은 네온 라인으로 감싸서 'The System Gauge'의 미학을 차용. 색상은 딥 블루 (#007FFF). | 마크 외곽선 |
| **카피 (헤드라인)** | "단순한 재미를 넘어, 검증된 안정성을 경험하세요." | 상단 배치 |
| **톤앤매너** | 매우 깔끔하고 권위적이며, 기술적 신뢰도를 시각적으로 명확하게 전달하는 미니멀리즘 강조.
