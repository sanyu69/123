# 웹사이트 메인 배너 최종 디자인 명세 (안 1: The System Gauge - MAX)

## 🎨 시각 시스템 적용
*   **스타일:** Neon\_Vibe, High Contrast, Cyberpunk Aesthetic
*   **주요 컬러:** Background: `#0A0A1A` (Deep Space Black), Accent: `#00FFFF` (Neon Cyan), Secondary Accent: `#FF00FF` (Electric Magenta).

## 🖼️ 레이아웃 및 구성 요소
1.  **배경:** Deep Space Black 배경에 미세한 네온 그리드 패턴을 은은하게 적용.
2.  **The System Gauge:** 화면 중앙에 거대한 원형 게이지를 배치. 게이지 내부의 채움 정도는 $R_{Stability}$ 데이터의 현재 상태를 반영하며, 최대치 도달 시 Cyan/Magenta 빛이 폭발하는 애니메이션 효과 명시.
3.  **핵심 메시지 (Primary Text):** "SYSTEM STABILITY: MAX" (Electric Magenta). 폰트 크기 최대로 중앙에 배치.
4.  **보조 정보 (Secondary Text):** "Real-time Stability Data Verified by Developer Team" (Pale White). 하단 중앙에 배치하여 신뢰도를 확보.

## ⚙️ 기술 요구사항
*   모든 요소는 고해상도 벡터 기반으로 디자인되어야 함.
*   애니메이션 효과(게이지의 빛나는 효과)를 포함하여 최종 이미지를 생성해야 함.
