# 웹사이트 메인 배너 최종 이미지 생성 명세 (Final Asset Specification)

## 🎯 목표: $R_{Stability}$ 지표 강조형 마케팅 소재

**사용 목적:** 웹사이트 메인 배너 (최고 인게이지먼트 확보)
**핵심 요구사항:** 'The System Gauge' 시각 시스템을 완벽히 구현하여 기술적 안정성을 프리미엄 가치로 포지셔닝.

### 🎨 Visual Style & Color Palette
*   **Style:** Neon Vaporwave / Dynamic Glow Effect (GLOW & Dynamic Effects 미학 극대화)
*   **Palette:**
    *   Primary Glow: `#00FFFF` (Neon Blue)
    *   Accent Glow: `#FF00FF` (Neon Magenta)
    *   Background Base: `#1A1A2E` (Deep Dark Navy/Black)
    *   Text/Gauge Fill: 밝은 네온 계열

### 📐 Layout & Composition
*   **Structure:** 중앙 집중형 Gauge 시각화가 메인.
*   **Element 1: The System Gauge:** 화면 중앙에 원형 또는 게이지 형태의 시각적 요소 배치. 이 게이지는 현재 시스템 안정성($R_{Stability}$) 상태를 동적으로 표현해야 함 (예: 채워진 정도, 빛의 강도 변화).
*   **Element 2: Stability Score:** $R_{Stability}$ 점수를 가장 크고 선명하게 중앙에 위치시킴. (예: **$R_{Stability}$: 98.5%**)
*   **Element 3: Headline:** 핵심 슬로건을 강조. (예: "The Ultimate System Stability")
*   **Typography:** 모든 텍스트는 네온 스타일의 산세리프 폰트를 사용하며, 광원 효과(Glow)가 적용되어야 함.

### ✨ Final Rendering Prompt (AI 이미지 생성용)
"A highly detailed, cinematic digital art banner for a website main header. The style must be 'Neon Vaporwave' with intense 'GLOW & Dynamic Effects'. Center the composition around a futuristic 'System Gauge' visualization that dynamically represents system stability ($R_{Stability}$). Use a deep dark background (`#1A1A2E`) illuminated by vibrant Neon Blue (`#00FFFF`) and Neon Magenta (`#FF00FF`). The score '$R_{Stability}$: 98.5%' must be the most prominent element, glowing intensely. Include subtle dynamic neon lines and futuristic UI elements to emphasize system stability as a premium feature."
